# 简介
ARM相关，以下所有汇编皆基于armv7
# 指令解析
```asm
EOR.W           R3, R3, R3,LSR#6 
```
对应c伪代码
```c
r3=r3^(r3>>6)
```
```asm
str     r3, [r2, r1, lsl #2] 
ldr     r3, [r2, r1, lsl #2] 
```

对应c伪代码

```c
r2[r1]=r3;//r2为32位整型数组
r3=r2[r1];
```
> 这里的汇编会有<<2操作，是因为整形占4字节，<<2也就是\*4,c里面的索引是按整形来数的，而汇编里是按字节来数的，所以需要\*4
