# 解BL锁
- 开机
- 开启开发者模式(设置-关于-软件信息-编译编号按7次)
- 开启OEM解锁
- 开启调试模式
- 关机(长按电源键和音量-键)
- 同时按住音量-和音量+键
- 等待几秒
- 插入USB并连接电脑
- 等待进入下载模式
- 进入下载模式后长按音量+键，然后按提示确认即可
- 之后等待重启手机
