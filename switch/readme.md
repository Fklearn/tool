# 购买方式
## 贝宝
> 优点：方便，安全 \
> 缺点：支持地区少
## 信用卡
> 优点：支持地区多（相对贝宝来说）\
> 缺点：有点不安全（自认为，毕竟信用卡不需要密码）
## 点卡
> 优点：支持全部地区（大概）\
> 缺点：贵点，还有余额问题导致无法换区（日本除外）\
> 注：点卡不会有假的
